# Mine_project_Numpy
# 📊 Student Marks Analysis Using NumPy

## 📌 Project Overview

The **Student Marks Analysis Using NumPy** project is a simple Python application that demonstrates how the **NumPy** library can be used to analyze student examination data efficiently. The project stores marks of five students across four subjects and performs various statistical calculations to generate meaningful insights.

This project is suitable for beginners who are learning **Python**, **NumPy**, and the basics of **Data Analysis**.

---

# 🎯 Objectives

* Learn how to create and manipulate NumPy arrays.
* Analyze student marks using NumPy functions.
* Calculate totals, averages, and percentages.
* Identify the top-performing student.
* Find the hardest subject based on class average.
* Filter students who scored above 80%.

# 📌 NumPy Functions Used

| Function         | Purpose                     |
| ---------------- | --------------------------- |
| `np.array()`     | Create arrays               |
| `np.sum()`       | Calculate total marks       |
| `np.mean()`      | Calculate average marks     |
| `np.argmax()`    | Find highest scorer         |
| `np.argmin()`    | Find lowest average subject |
| Boolean Indexing | Filter students above 80%   |
